# Pudser Support Hub – GitHub Pages / PWA

Læg alle filer i roden af dit GitHub Pages-repo.

Filer:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png
- icon-192-maskable.png
- icon-512-maskable.png

## GitHub Pages
1. Upload filerne til repoet.
2. Gå til Settings -> Pages.
3. Vælg branch (typisk `main`) og root (`/`).
4. Gem.

## Installering
Når siden er live på HTTPS via GitHub Pages, kan den installeres som app på understøttede enheder/browsere.
